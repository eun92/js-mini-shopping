# 프로젝트<br>

### 드림코딩 미니 쇼핑몰<br>

순수 자바스크립트를 이용하여 만들어보는 아주 간단한 미니 쇼핑몰
<br><br>

# Skills

1. html
2. css
3. javascript

- 간단한 프로젝트이기 때문에 어떤 프레임워크 없이 순수 html, css, js를 이용<br>
- 순수 javascript를 익히는 데에 중점을 두며 개발
  <br><br>

# 주요 기능

1. 전체 아이템 목록 출력
2. 타입별 목록 필터(정렬)
